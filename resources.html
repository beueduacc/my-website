Resources

  • Purchasing a Raspberry Pi
    • Note: The Raspberry Pi site is the official source for Apporoved Resellers. Devices can be found on sites such as Ebay or Amazon, but beware of these as they are often fake or come with preinstalled malware!
  • The Raspberry Pi Foundation
    • This is a great place to keep up with Raspberry Pi news.
  • Raspberry Pi-Spy
    • Pi-Spy is a community run site full of blogs, tutorials, and project ideas. This is a great resource for anyone interested in hacking a Raspberry Pi device.
  • Ubuntu for Raspberry Pi
    • Ubuntu is one of the the most popular Linux distros. It's very beginer friendly and with a little work can be made to look and function like a Windows or Mac OS.
  • RetroPie
    • RetroPie is an emulation software that allows you to run games from a wide range of retro consoles from the Atari 2600 and Apple II, all the way to the PlayStation 2 and Nintendo DS.
